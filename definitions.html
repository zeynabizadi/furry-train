Html چیست؟


عبارت HTML ( اچ تی ام ال ) مخفف Hyper Text Markup Language معنی زبان نشانه گذاری فوق متن است .


HTMl زبان استاندارد طراحی صفحات وب است و کليه کدهای صفحه اعم از طرف سرور و طرف مشتری در نهايت به کدهای HTML تبديل شده و توسط مرورگر نمايش داده می شوند.

به عبارت دیگر مرورگر ها هیچکدام از کدها و کنترل های سمت سرور همچون کدهای asp و php را نمیشناسند و کد قابل فهم برای آنها اچ تی ام ال می باشد.

کامپایلر های زبان های برنامه نویسی سروری در نهایت کد های خود را برای نمایش به کد اچ تی ام ال تبدیل میکنند و برای مرورگر میفرستند تا به کاربران نمایش داده شود.



HTML یک زبان نشانه گذاری است ، به اين معنی که بخش های مختلف توسط اجزايی به نام تگ از هم جدا شده ، که هر کدام دارای کاربرد و خواص مربوط خود هستند . اين تگ ها به مرورگر اعلام می کنند که هر بخش از صفحه چه نوع عنصری است و بايد به چه صورت نمايش داده شود . 
در يک صفحه HTML می توان انواع عناصر از قبيل متن ، تيتر ، عکس ، جدول  و ... را قرار داد ،
 که برای هر عنصر بايد از تگ مربوط به آن استفاده کرد . صفحات HTML فقط از کد ها که به صورت متن هستند تشکيل شده اند

تعریف تگ ها


تگ bdi : برای تعریف جهت یک محتوا ( مثلا برای گذاشتن متن انگلیسی در متن فارسی )

تگ canvas : یک ظرف برای نگهداری اشکل گرافیکی ایجاد شده توسط اسکریپت

تگ figure : برای گذاشتن یک محتوای مستقل در یک صفحه وب (مثله نمودارها،تصاویر و..)

تگ input : برای قرار دادن کنترل های ورودی در یک فرم

تگ svg : یک ظرف برای نگهداری اشکال گرافیکی ایجاد شده توسط SVG

تگ strong : یک متن را برجسته یا کلفت میکند. ( با ارزش محتوایی بالا )

تگ s: برای خط کشیدن بروی یک متن (یعنی اون متن تاریخ اعتبارش تموم شده)

تگ output :برای نمایش خروجی یک عمل انجام شده (مثله عمل ماشین حساب)

تگ time : برای ایجاد یک زمان – (تاریخ و ساعت) در یک صفحه وب

تگ meter : میتوان یک گیج (Gauge) اندازه گیری را در یک صفحه وب ایجاد کرد

تگ keygen : برای تولید یک جفت کلید یا کد رمزنگاری شده در سمت کاربر و سرور

تگ title : برای ایجاد عنوان یک صفحه وب در مرورگرها و موتورهای جستجوگر



چند مورد برتری Html5 بر HTML چیست؟

1- شامل جزئیات مدل‌های پردازش است تا پیاده سازی سازگارتری داشته باشد. 2- نشانه گذاری برای اسناد در دسترس را گسترش، بهبود و توجیه می‌کند. 3- نشانه گذاری و رابط برنامه نویسی کاربردی را برای برنامه‌های پیچیده تحت وب معرفی می‌نماید.
